# Planning Workflow

This folder holds draft content before it is promoted to `data/trips/`.

## Process
1. Plan a trip here in markdown
2. Get approval
3. Convert to JSON → `data/trips/{slug}.json`
4. Add trip card to `index.html`
5. Build out full day detail in `trips/{slug}/index.html` (current approach)
   OR wire up `trip.html` template (future data-driven approach)
